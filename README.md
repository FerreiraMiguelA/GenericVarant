# GenericVarant
Generic Varant to run Coding test

1º Install Vagrant https://www.vagrantup.com/;
2º Install Virtual Box https://www.virtualbox.org/wiki/Downloads;
3º Edit puphpet/config.yaml file and change synced_folder:source to your CodingTest project path;
4º Iniciate vagrant machine (vagrant up);
5º Test it on browser using ip 192.168.56.101 or codingtest.dev (add it to your hosts file);
